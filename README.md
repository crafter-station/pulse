# Pulse

Shipping tracker for [Crafter Station](https://github.com/crafter-station).

Tracks pushes to `main` across all org repos.

## Stack

- Vite + React + TypeScript
- Tailwind CSS
- Trigger.dev (webhooks)
- Neon (postgres)

## Dev

```bash
bun install
bun dev
```
